# OIBSIP Task 1 - Landing Page

## Description

This project is a responsive product landing page developed as part of the Oasis Infobyte Web Development and Designing Internship. It demonstrates modern frontend design principles with a clean user interface, responsive layout, and structured content sections to provide an engaging user experience.

## Features

- Responsive design
- Hero section
- Navigation bar
- Features section
- Testimonials section
- Call-to-action section
- Cross-device compatibility

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts

## Outcome

Successfully developed a responsive landing page with a clean design and improved user experience while strengthening frontend development skills.
